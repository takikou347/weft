## E2E(Playwright・6スペック)

### e2e/rls-isolation.spec.ts(必須の分離テスト)

| ケース | 検証 |
|---|---|
| Aが記録を作成し自分では見える | UI通し |
| BのUIにAの非共有アイテムが出ない | 一覧・空状態 |
| API直叩きでも見えない | supabase-js直接: 一覧0件/ID直指定0件/UPDATE無効/不正共有拒否/スペース・プロフィール分離 |
| 匿名 | UIは/loginへ・APIは0件 |

### e2e/p2-personal-core.spec.ts

予定作成→その日ページ / 派生導線で日記→**自動双方向リンク**(両方向から確認)/ 収支→月次集計反映 / カレンダー→今日のページ

### e2e/p3-sharing.spec.ts

グループ作成→招待状発行→日記を差し出す / Bが招待から参加→回覧板に見える・**未共有は見えない**→コメント・スタンプ→個人こよみにレイヤー / **第三者Cには404・一覧にも出ない** / 取り下げでBから消える・**Aには残る**

### e2e/p4-group.spec.ts

装飾つき日記(はんこ表示確認)→写真アップロード(実圧縮パス)→差し出し / なかまがアルバムで写真+日記閲覧(署名URL) / 立替記録→精算案(12,000円)→精算済み化

### e2e/p5-org.spec.ts

組織→しごと→つとめ→予算→おぼえの作成通し / **組織なかまはしごと404(§6.3)** / 参加後: 閲覧可・他人のつとめは動かせない・自分のつとめのステータス変更・ダッシュボード表示 / 部外者は組織もしごとも404

### e2e/p6-release.spec.ts

規約・プライバシーが未ログインで読める / 共有・コメントで便りが届く→全既読 / **退会→ログイン不可**
