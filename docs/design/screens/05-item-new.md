# 05. 記す(アイテム作成)

- URL: `/items/new?type=diary|event|expense|task&date=YYYY-MM-DD&link=<itemId>`
- アクセス: 要ログイン / 対応項番: F-03-2, F-04-1/2/4, F-05-1, F-03-4/F-09-3(link)

## 共通項目

| No | 項目 | 種別 | 必須 | 内容・表示条件 |
|---|---|---|---|---|
| 1 | 種別タブ(日記/予定/収支/つとめ) | リンク | − | 常時。選択中は藍地。`date`/`link` を引き継いで切替 |
| 2 | 見出し+注記 | | − | 種別ごと(例: 日記=「日記を記す/記したことは、あなたにしか見えません。」)**デフォルト非公開の明示** |
| 3 | 日付 | date | ○ | 既定=URLの`date`(なければ今日) |
| 4 | エラー文言 | alert | − | 検証失敗時のみ(下表) |
| 5 | もどる | リンク | − | → `/days/{date}` |
| 6 | 帳面に記す | ボタン(藍) | − | 送信中「記しています…」 |
| 7 | link_to(hidden) | hidden | − | `?link=` 指定時のみ。**保存時に自動で双方向リンクを作成**(F-03-4/F-09-3) |

## 種別ごとの項目

### 日記(type=diary)

| 項目 | 種別 | 必須 | 内容 |
|---|---|---|---|
| 題 | input | −(※) | ※題か本文のどちらか必須 |
| 本文 | textarea(8行) | −(※) | |
| 用紙 | radio | − | 白紙(既定)/便箋/方眼/生成り(F-04-4) |
| しるしのはんこ | radio | − | なし(既定)/花丸/よき/拝見/感謝 |

### 予定(type=event)

| 項目 | 種別 | 必須 | 内容 |
|---|---|---|---|
| 題 | input | ○(※題か本文) | |
| 終日 | checkbox | − | ONのとき保存時に時刻を破棄 |
| はじまり〜おわり | time×2 | − | |
| 場所 / おぼえがき | input / textarea | − | |

### 収支(type=expense)

| 項目 | 種別 | 必須 | 内容 |
|---|---|---|---|
| 出入り | radio | ○ | 支出(既定)/収入 |
| 金額(円) | number | ○ | 1以上の整数 |
| 費目 | input+datalist | − | 自分の費目一覧を候補表示・**自由入力も可**。空→「その他」 |
| 摘要 | input | − | |

### つとめ(type=task)

| 項目 | 種別 | 必須 | 内容 |
|---|---|---|---|
| 題 | input | ○(※題か本文) | |
| すすみ具合 | select | − | これから(既定)/とりくみ中/しあげた |
| 本文 | textarea(3行) | − | |

## 処理・検証

| 操作 | 検証(エラー文言) | 成功時 |
|---|---|---|
| 帳面に記す | 日記・予定・つとめ:「題か本文のどちらかを記してください。」/ 収支:「金額は1円以上の整数で入れてください。」/ 種別不正:「種別が正しくありません。」 | items INSERT(origin=個人スペース=**非公開で生まれる**)→ link指定時はリンクも作成 → `/days/{日付}` へ |

## パターン

| パターン | 挙動 |
|---|---|
| type未指定/不正 | 日記タブになる |
| date不正 | 今日に正規化 |
| 派生導線から(link付き) | 保存後、元アイテムと自動で「結びついた記録」になる |
| 検証エラー | alert表示・入力値は保持 |
