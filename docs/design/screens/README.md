## 全画面共通仕様

### 共通ヘッダー(ログイン後の全画面)

| 項目 | 内容 | 表示条件 / 処理 |
|---|---|---|
| ロゴ「Weft」 | 明朝体・トラッキング広め | 常時。タップ→ `/`(帳面) |
| 便り | 通知一覧へのリンク | 常時。**未読が1件以上あるとき右肩に藍色ドット**(aria-label="未読N件")。タップ→ `/notifications` |
| 「◯◯ さん」 | プロフィールの表示名 | 常時。タップ→ `/account` |
| とじる | ログアウト | 常時。Server Action logout → `/login` |
| ナビ(こよみ/帳面/家計/つながり) | 主要4ページ | 常時。各 `/calendar` `/` `/expenses` `/spaces` |

### デザイントークン

| トークン | 値 | 用途 |
|---|---|---|
| washi | #f6f3ea | 地(背景) |
| paper | #fffdf7 | 面(カード・表) |
| sumi / usuzumi | #333230 / #6b675f | 文字 / 補助文字 |
| keisen | #ddd6c7 | 罫線・枠 |
| ai / ai-deep | #3f5d7d / #32485f | 差し色(ボタン・リンク)/ hover・エラー文言 |
| 書体 | Zen Kaku Gothic New(本文)/ Shippori Mincho(見出し・ロゴ) | 行間1.9 |

### 共通パターン

| パターン | 挙動 |
|---|---|
| 未ログインで非公開URLへ | `/login?next=元パス` へ302。ログイン成功後に元パスへ戻る |
| 権限のないリソース(RLSで0件) | 404ページ(存在自体を隠す)。[06参照](./06-item-detail.md) |
| フォームエラー | フォーム内に `role="alert"` の文言(ai-deep色)。入力値は保持 |
| 送信中 | ボタンをdisabled+文言変化(例:「記しています…」) |
| 一覧が21件以上 | 「まえの頁 / n / N 頁 / つぎの頁」のページネーション(1頁20件、アルバムのみ30件) |
| 空状態 | 中央寄せの薄墨文言+次の行動の誘い(各画面の空パターン参照) |
| 日付・金額表記 | 「2026年7月20日」「12,800円」(収入は + と藍色) |
