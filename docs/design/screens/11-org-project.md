# 11. つとめ先(組織)・しごと(プロジェクト)

- アクセス: メンバーのみ。**組織のなかまでも、しごとに参加していなければ中身は見えない**(§6.3)
- 例外: 組織のowner/adminは配下のしごとの「名前と予実・状態」だけ一覧で見える(F-08-5)

## 11-1. しごと一覧(組織ダッシュボード)

- URL: `/spaces/{orgId}/projects` / 対応項番: F-02-2, F-08-5

![projects](../images/org-projects.png)

| No | 項目 | 内容・表示条件 |
|---|---|---|
| 1 | しごとリスト | 配下プロジェクト。名前(**参加中のみリンク**・非参加は薄墨の文字のみ)+状態ラベル(支度中/とりくみ中/しあげた) |
| 2 | 予実行 | 「予算 N円 / 実績 N円 / 残り N円」。**実績は参加しているしごとのみ**(非参加は「実績は参加者のみ閲覧できます」=§6.3) |
| 3 | あたらしいしごと | **組織のowner/adminのみ**。名前入力+「はじめる」→ RPC create_project(作成者がしごとのowner)→ 新しいしごとの回覧板へ |
| 4 | 空状態 | 「まだしごとはありません。」 |

## 11-2. つとめ(タスク)

- URL: `/spaces/{projectId}/tasks` / 対応項番: F-08-2

![tasks](../images/project-tasks.png)

| No | 項目 | 内容・表示条件 |
|---|---|---|
| 1 | 状態セクション | これから/とりくみ中/しあげた の3段(件数つき)。0件は「ありません。」 |
| 2 | つとめ行 | 題(→共有アイテム画面)+「M月D日まで」+担い手(未定は「きめず」) |
| 3 | 移動ボタン | **作成者か担い手のみ**「◯◯へ」×2(他の2状態へ)。RPC update_task_status(**本文は書き換え不可**の唯一の例外操作) |
| 4 | つとめを書き出す | メンバー全員。なにをするか(必須100字)/期限(date)/担い手(select: きめず+メンバー)→ 作成と同時にしごとへ共有。担い手には便り(task_assigned) |

## 11-3. 予実

- URL: `/spaces/{projectId}/budget` / 対応項番: F-08-1, F-08-3, F-05-5

![budget](../images/project-budget.png)

| No | 項目 | 内容・表示条件 |
|---|---|---|
| 1 | 三面サマリー | 予算(projects_meta.budget_total)/ つかい(実績=**このしごとへ差し出された収支の合算**・RPC)/ 残り(負はあかね色) |
| 2 | 損益行 | **収入実績があるとき**「はいり(実績): N円 / 損益: N円」(F-05-5) |
| 3 | 実績の注記 | 常時「実績は、このしごとへ差し出された収支の合算です。」 |
| 4 | 費目別の予算 | budgets一覧(費目+金額)。**owner/adminのみ**追加フォーム(費目20字+金額)と×(削除) |
| 5 | しごとの手入れ | **owner/adminのみ**(F-08-1): すすみ具合(select)/はじまり・おわり(date)/予算総額 →「しまう」 |

## 11-4. おぼえ(ナレッジ)

- URL: `/spaces/{projectId}/docs` / 対応項番: F-08-4, F-08-6 / `?page=N`(20件)

| 一覧+作成 | 詳細(コメント可) |
|---|---|
| ![docs](../images/project-docs.png) | ![doc-detail](../images/project-doc-detail.png) |

| No | 項目 | 内容・表示条件 |
|---|---|---|
| 1 | おぼえ書きリスト | 新しい順。題+本文2行+日付 → 共有アイテム画面(コメント・スタンプ可=議論の場) |
| 2 | 空状態 | 「まだおぼえ書きはありません。」 |
| 3 | おぼえ書きをのこす | 題(必須100字)+本文(必須)→ 作成と同時にしごとへ共有 |
| 4 | エラー文言 | `?error=1`「のこせませんでした。題と本文の両方を入れてください。」 |

※ おぼえとつとめ・予定の双方向リンク(F-08-6)は、作成者が自分のアイテム詳細(06)の「結びつける」から張れる。

## パターン

| パターン | 挙動 |
|---|---|
| 組織メンバー(非owner/admin)が /projects | しごと一覧は見えるが「あたらしいしごと」非表示・非参加のしごとは実績非表示 |
| 組織メンバーがしごとのURL直打ち | 404(参加するまで中身は存在ごと不可視=§6.3) |
| member が予実を開く | サマリー・予算は閲覧可。追加フォーム・手入れは非表示 |
| 担い手でも作成者でもない人のつとめ | 移動ボタン非表示(RPC側でも拒否) |
